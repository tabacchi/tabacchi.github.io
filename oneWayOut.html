One Way Out


By: Thomas Tabacchi, Jajuan Webster and Lab Nguyen

Summary

An endless typing game aimed at teaching people how to learn sign language. In this game you must type out the name seen above the slime and which is a mix between english letters and the sign language letters so the player will eventually learn. This gets progressively harder the longer that the player survives until all the letters are sign language.
