Ecommerce Website


By: Thomas Tabacchi and Joseph Han

Summary

Using Vue, Javascript, HTML and CSS we created a template for an ecommerce website. You can enter the product that you wish to sell on this website and it would save it to database.

What I Worked On

  • Utilizing Firebase to save products
  • Using Javascript to save the fields in each step and prints them on the final screen
  • Using the Vue, Vuex, and Vue router templates to create these pages
  • Creating part of a website in HTML and CSS
